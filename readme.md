# Project Overview

This project is developed and tested under **Python 3.12**.

It supports two execution modes: **simulation mode** and **real-world mode**.

## Running the Project

### Simulation Mode

To run the project in simulation mode, execute the following command in the terminal:

```bash
python qt_gui.py -s
```
